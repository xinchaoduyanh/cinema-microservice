# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project and community a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful toward others
- Giving and accepting constructive feedback
- Demonstrating empathy and understanding

Examples of unacceptable behavior include:

- Harassment or derogatory comments
- Trolling or insulting comments
- Publishing others' private information without consent

## Enforcement Responsibilities

Project maintainers are responsible for enforcing the Code of Conduct.

## Scope

This Code applies within project spaces and also applies when representing the project publicly.

## Enforcement

Instances of abusive, harassing, or unacceptable behavior may be reported by opening an issue or contacting the maintainers.

Reports will be handled confidentially and fairly.

### How to Report

Please report instances of abusive, harassing, or otherwise unacceptable behavior by contacting the project maintainer(s) directly at:

**nestturbo.maintainer@gmail.com**

All reports will be reviewed and investigated, resulting in a response that is deemed necessary and appropriate to the circumstances.
